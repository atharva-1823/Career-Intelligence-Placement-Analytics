# Career Intelligence & Placement Analytics Platform

## Problem Statement

Students often apply to jobs without understanding whether their skills match industry requirements. This project analyzes job descriptions, extracts in-demand skills, identifies skill gaps, and provides personalized placement insights.

## Objectives

- Collect and analyze job descriptions
- Extract in-demand skills
- Perform skill-gap analysis
- Generate placement readiness insights
- Build interactive dashboards

## Tech Stack

- Python
- Pandas
- MySQL
- Power BI
- Streamlit
- Git/GitHub
